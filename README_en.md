# <center>Google Translate Page</center>
[Google.com](https://google.com) is the most visited website in the world, so [translate.google.com](https://translate.google.com) is translated by Google. This repository is Call the Google Translate API to translate the entire webpage.
# Code
## Translate Page To German
```html
<a href="javascript: void(window.open('http://translate.google.com/translate?u='+escape(location.href)+'&amp;langpair=en%7Cde', 'English_to_German', 'resizable,scrollbars,status'))">网页翻译为德语(Translate Page To German)</a>
```
